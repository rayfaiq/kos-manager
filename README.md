# Kos Manager

Aplikasi pengelolaan keuangan kos sederhana dan modern.

## Fitur

- ✅ Daftar kos + harga
- ✅ Indikator isi/kosong
- ✅ Pengingat pembayaran
- ✅ Total pendapatan otomatis
- ✅ Data tersimpan di browser

## Cara Menggunakan

1. Buka aplikasi di browser
2. Centang ruangan yang sudah ditempati
3. Klik "Simpan" untuk menyimpan perubahan
4. Data akan tersimpan otomatis

## Deployment

Dilayani oleh Vercel → [https://kos-manager-psi.vercel.app](https://kos-manager-psi.vercel.app)
